------------------------------
Reverse Oreo Team Present
------------------------------
Система автоматизированного управления ядерным реактором

---

## О проекте

Reverse Oreo Team Present — это десктопное приложение для мониторинга и автоматизации контроля параметров ядерного реактора. Программа предназначена для оперативного отслеживания критических показателей, автоматической поддержки безопасного режима работы и регистрации инцидентов.

---

## Функциональные возможности

### Мониторинг параметров

Приложение отображает четыре основных параметра состояния реактора:

- Температура активной зоны
- Уровень воды в первом контуре
- Уровень радиационного фона
- Интегральный рейтинг системы

Для каждого параметра определён допустимый диапазон значений. Выход за его пределы визуально маркируется.

### Автоматизация

Система поддерживает три автоматических механизма стабилизации:

- Охлаждение — снижение температуры при перегреве
- Добавление воды — поддержание необходимого уровня теплоносителя
- Снижение радиационного фона — активация штатных систем очистки

### Реагирование на инциденты

Интерфейс дополнен блоком отображения актуального инцидента. Система позволяет отслеживать текущее критическое событие и реагировать на него в ручном или автоматическом режиме.

### Ограничения и инструменты

Для предотвращения избыточной нагрузки на систему введён таймаут на ручной долив воды — 15 секунд после каждого использования. Также доступна кнопка вызова окон с графиками для долгосрочного наблюдения за динамикой параметров.
