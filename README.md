---------------------------------------
REVERSE OREO TEAM PRESENT
---------------------------------------
Данное программное обеспечение представляет собой систему автоматизированного управления ядерным реактором.
В главном окне программы реализована возможность отслеживания за параметрами ядерного реактора.
На данный момент отображается три параметра: температура, вода, радиация.
Существует диапазон значений приемлимый для работы реактора.
Для корректной работы добавлена система автоматизации - охлаждение, добавление воды и снижение радиацонного фона.
